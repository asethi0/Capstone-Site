About

Securely Sign In

Securely sign into our application through your Amazon account. After your initial sign in you can use your username and password. To ensure that no one can use the application when you don't have your phone, tokens are refreshed every 3 hours of inactive use.

Installation Instructions

Windows

Android

Download Android Studio

  • Download and install Android Studio.
  • Clone the project from the Github repository: https://github.com/zacharynoel/Capstone
  • Make sure to install Android 28 under Settings.
  • Sync gradle scripts.
  • Choose an emulator for testing.
  • If you have an AMD processor, use an external emulator (Bluestacks or Genymotion).
  • Android Studio environment is set up.

MySQL Database Setup

    Download MySQL Community: https://dev.mysql.com/downloads/mysql/
    During installation, select to start MySQL on startup.
    Ensure that your root password is "root".
    Download our services repository: https://github.com/mtriau/CreamCrusaders-Services
    Run HandmadeApiApplication.java from the services repository. Keep this running while testing the application on Android Studio.
    Within the services repository is a dumpfile, use this for the required database tables.
    Check under services.msc that MySQL #.# is currently running (#.# for the installed version number)
    If MySQL server is running under services, HandmadeApiApplication.java is running, and the tables are setup, the MySQL backend setup is complete. Run the app in android studio to begin testing.
